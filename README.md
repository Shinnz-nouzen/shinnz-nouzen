# The Siblings Mart & Café

This repository contains a simple static website for The Siblings Mart & Café in Tboung Khmum.

## GitHub Pages Setup

1. Create a new GitHub repository and push this folder as the repository root.
2. In GitHub, go to **Settings > Pages**.
3. Select the branch you want to publish from (usually `main` or `master`).
4. Set the folder to `/ (root)`.
5. Save and wait a few minutes for the site to become available.

Your website will be served from `https://<your-username>.github.io/<repository-name>/`.

## Files

- `index.html` — the homepage for the website.
- `.nojekyll` — ensures GitHub Pages serves this file without Jekyll processing.
